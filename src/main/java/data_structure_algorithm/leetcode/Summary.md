# 2. 位运算
## 2.1. 总结
#### 2.1.1. lowbit(最低有效位)：x & (-x)。注意：补码表示；最小的负数会溢出。 -> res = x == Integer.MIN_VALUE ? x : x & (-x)
#### 2.1.2. 求余：任意正整数对2^n求余，等于该数与(2^n-1)按位与。X % 2^n  = X & (2^n - 1)
#### 2.1.3. 位与：n & (n-1)消除n末尾的1
#### 2.1.4. 位或：计算所有位的和
#### 2.1.5. 移位：左移实现倍增（快速乘法）；右移实现倍减（快速除法）
#### 2.1.6. 异或：相同数字的异或结果为0，任何数字与0异或的结果为该数字本身

# 15. 全局搜索
## 15.1. 总结
### 15.1.1. DFS(Depth First Search)深度优先搜索：借助递归实现，可实现全局搜索，一般用于找所有路径(回溯)。
#### 15.1.1.1. 当只需要判断路径是否存在时(找一条可行路径)，递归函数可以设计成返回值为布尔值的短路形式；
#### 15.1.1.2. 当要找所有路径时，不可以设计成返回值为布尔值的短路形式，需要遍历所有的解空间；
